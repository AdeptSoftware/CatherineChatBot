# Хранилище данных

Содержимое файла data.json (в корневой папке):
{
	"dst":		"Catherine1.5/",		// В какую папку в Яндекс.Диск'е загружаем
	"src":		"local_storage/",		// Локальная папка для загрузки на Яндекс.Диск
	"out":		"logs/main.txt",		// Куда выводить логи приложения
	"token":	"YandexDiskToken"
}
// Файл data.json используется только в app.py
// Поэтому обозначения ключей весьма условные

Структура хранилища (root = data["dst"]):
Обязательные файлы:
root/logs/main.txt						// Может быть пустым. Хранит данные о ошибках
root/auth.json							// Входные данные в другие API
root/app.json							// Настройки приложения

Необязательные файлы:
root/backup/nicknames/*.json			// База ников
root/userdata.json						// Данные пользователей
root/events.json						// События, оповещения и тд
root/topics.json						// Откуда брать новые/обновлять ники
